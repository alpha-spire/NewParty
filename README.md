Le projet Party est une application mobile dont l’objectif principal est de simplifier le partage, l’organisation et la sauvegarde de photos entre plusieurs utilisateurs pour un évènement donné.
L’application repose sur un principe simple : un utilisateur peut créer un événement en définissant différents paramètres (titre, date, localisation, participants). À partir de cet événement, deux éléments sont générés automatiquement :
-	Un album photo collaboratif
-	Un espace de discussion (chat de groupe)

Pendant toute la durée de l’événement, les participants peuvent ajouter des photos à l’album associé. Cela permet de centraliser tous les souvenirs liés à un moment donné (soirée, voyage, anniversaire, etc.) dans un seul espace, évitant ainsi la dispersion habituelle des contenus. 
Le chat intégré permet également aux utilisateurs d’échanger directement dans le contexte de l’événement, renforçant la cohérence entre communication et partage de médias.
